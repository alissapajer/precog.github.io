AppendFormat Precog.Client.Format.Formats.JSON = new AppendFormat("application/json") [static]

The standard JSON type for single JSON elements or JSON arrays of elements.

The JSON format indicates either a root-level JSON element (literal or object), or a JSON array of elements. In the latter case, each element of the root-level array is ingested as a separate event. If you actually want to ingest a JSON array as an event, us the JSONStream format.

In JSON format, root-level JSON arrays are treated as a collection of JSON elements to be processed as separate events. This means that the following code produces identical ingests. 

                    client.AppendRaw("/test", "{\"test\":1}", Formats.JSON)
                    client.AppendRaw("/test", "[{\"test\":1}]", Formats.JSON)
AppendFormat Precog.Client.Format.Formats.JSONStream = new AppendFormat("application/x-json-stream") [static]

A JSON type indicated concatenated JSON elements.

The JSONStream format indicates a concatenation of distinct JSON objects, arrays, or literals in a stream, without delimiters between elements.

The following is an example of a JSON stream: 

                          {"test":1}{"test":2}{"foo":{"bar":{"baz":3}}}
