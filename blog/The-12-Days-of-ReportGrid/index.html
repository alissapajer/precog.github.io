The 12 Days of ReportGrid

We've gotten into the holiday spirit lately and decided to share our favorite ReportGrid charts in a fun campaign, The 12 Days of ReportGrid. In a few hours, we created custom charts using the ReportGrid API and the Precog analysis platform. Click on a present to reveal a new chart for each day.

The ReportGrid visualization library is built in HTML5 and has a plethora of customizable options:

    • Hundreds of pre-made color palettes (if thats not enough, make your own)
    • Labels, axes, rules and titles easy to change and format
    • Customizable tooltips to display additional information about your data
    • Style with CSS
    • Many, many options to create unique visualizations

To see how easy it is to get started, check out a variety of code snippet examples that can be embedded directly into an HTML page. Just make sure to sign up on The 12 Days of ReportGrid page to get access to the free API.

One way to make use of ReportGrid visualizations is to use Precog for your backend data analysis. Precog provides a unified solution to embedded reporting by hosting the data, providing an interactive development environment to write queries in and making it easy to export query results into beautiful visualizations. For a step-by-step guide to quickly get embedded reporting in your application, see this How To in our Developer Center. If you need help at any point, just contact us and we'll assist with anything from the integration process to developing queries. Have fun charting!
